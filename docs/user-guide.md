## 界面布局

FuFanAI采用类微信PC端的三栏布局：

```
┌─────────────────────────────────────────────────────────┐
│  顶部工具栏：Logo + 全局搜索 + 通知 + 个人设置           │
├────────┬──────────────────────┬────────────────────────┤
│        │                      │                        │
│  📁    │     项目列表         │     详情区域           │
│ 项目   │                      │                        │
│        │     员工列表          │     工作区             │
│ 👤     │                      │                        │
│ 员工   │     市场内容          │     对话               │
│        │                      │                        │
│ 🏪     │     设置项            │     看板               │
│ 市场   │                      │                        │
│        │                      │                        │
│ ⚙️     │                      │                        │
│ 设置   │                      │                        │
└────────┴──────────────────────┴────────────────────────┘
```

| 区域 | 说明 |
|------|------|
| 左侧导航 | 4个模块切换 |
| 中间列表 | 当前模块内容列表 |
| 右侧详情 | 详细内容、工作区、对话 |

## 项目管理

### 创建项目

1. 点击「新建项目」
2. 输入项目名称和描述
3. 选择项目类型（可选）
4. 开始使用

### 项目功能

- **群聊** — 与AI团队实时对话
- **看板** — 可视化任务进度
- **工作区** — 项目文件存储和管理
- **人工审核** — 关键节点需确认

### 使用示例

```
你：帮我开发一个外卖App

AI团队：
[外卖App负责人]: 收到！正在启动项目流程...
[产品专家]: 开始进行需求调研
[开发专家]: 开始代码开发
...

你可以在看板上实时看到所有进度
```

## 员工管理

### 数字员工类型

| 类型 | 说明 |
|------|------|
| 管理型Agent | 项目负责人，调度专家完成任务 |
| 专家型Agent | 执行具体任务，如开发、测试、运维等 |

### 管理员工

- 查看员工信息
- 管理员工技能
- 与员工单聊
- 调整员工配置

## 设置

### 功能模块

| 设置项 | 说明 |
|--------|------|
| 技能管理 | 管理你的自定义技能 |
| 工具管理 | 配置外部工具接入 |
| API配置 | 设置AI服务API Key |
| 基础设置 | 语言、主题等 |

### 技能(Skill)

技能是数字员工的工作能力定义，包括：
- 输入声明
- 执行逻辑
- 输出声明
- 异常处理

## 核心概念

### 三层信息架构

| 层级 | 说明 |
|------|------|
| Agent私有记忆 | 每个Agent的长期记忆 |
| 项目工作区 | 项目相关的共享文件 |
| 对话上下文 | 当前对话的短期记忆 |

### 群聊模式

用户感知是一个AI团队在帮他工作：
- 每个Agent发言带有头像和名称
- 管理Agent主要与用户沟通
- 专家Agent进展通过管理Agent同步

# 故障排查

本指南帮助你解决使用中的常见问题。

---

## 启动问题

### 应用无法启动

**症状：** 双击图标没有反应

**解决方法：**

1. **检查系统要求**
   - 确认操作系统：Windows 10+ / macOS 11+ / Ubuntu 20.04+
   - 确认内存：最低8GB

2. **检查依赖**
   - Windows：安装 [.NET 6.0 Runtime](https://dotnet.microsoft.com/download/dotnet)
   - Linux：安装 GTK 3.0

3. **查看日志**
   - 应用目录下的 `logs/` 文件夹
   - 搜索错误信息

4. **重新安装**
   - 卸载后重新下载安装

---

### macOS提示"无法打开"

**症状：** Gatekeeper阻止运行

**解决方法：**

1. 系统偏好设置 → 安全性与隐私
2. 点击「仍要打开」
3. 或者：右键点击应用 → 选择「打开」

---

## 连接问题

### API连接失败

**症状：** 提示"无法连接到AI服务"

**解决方法：**

1. **检查网络**
   - 确认网络连接正常
   - 尝试访问AI服务商网站

2. **检查API Key**
   - 确认Key已正确配置
   - 检查Key是否有效/未过期

3. **检查配置**
   - 设置 → API配置
   - 确认API地址正确

---

### 超时错误

**症状：** 任务执行超时

**解决方法：**

1. 复杂任务适当增加超时时间
2. 检查网络稳定性
3. 简化任务描述重试

---

## 功能问题

### 任务执行失败

**症状：** AI团队任务失败

**解决方法：**

1. **查看错误信息**
   - 在对话中查看AI的回复
   - 查看工作区的错误日志

2. **检查输入**
   - 确认任务描述清晰
   - 确认必要文件已上传

3. **重新执行**
   - 简化任务描述
   - 分步骤执行

---

### 记忆丢失

**症状：** 每次启动都要重新解释背景

**解决方法：**

1. **检查存储**
   - 确认工作区文件存在
   - 检查存储空间充足

2. **项目设置**
   - 确认项目配置正确
   - 重新绑定项目

---

### 看板不更新

**症状：** 看板上任务状态不更新

**解决方法：**

1. 刷新页面
2. 重启应用
3. 检查任务执行状态

---

## 性能问题

### 运行缓慢

**症状：** 界面响应慢、卡顿

**解决方法：**

1. **检查内存**
   - 关闭其他占用内存的程序
   - 确认可用内存≥4GB

2. **清理缓存**
   - 设置 → 清理缓存

3. **检查存储**
   - 确认硬盘空间充足

---

### CPU占用高

**解决方法：**

1. 检查是否有任务在执行
2. 减少同时运行的任务数
3. 更新到最新版本

---

## 数据问题

### 工作区文件丢失

**解决方法：**

1. 检查文件是否被误删
2. 查看回收站
3. 从版本历史恢复（如有）

---

### 无法保存

**症状：** 保存操作失败

**解决方法：**

1. 检查存储空间
2. 检查文件权限
3. 尝试另存为

---

## 卸载与重装

### 完全卸载

**Windows：**
```
控制面板 → 程序和功能 → 卸载 FuFanAI
删除 AppData\Local\FuFanAI 目录
```

**macOS：**
```
删除 Applications/FuFanAI
删除 ~/Library/Application Support/FuFanAI
```

**Linux：**
```bash
sudo apt remove fufanai
rm -rf ~/.config/FuFanAI
```

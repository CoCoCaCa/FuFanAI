# 行为准则

## 我们的承诺

为了营造一个开放、友好的社区环境，我们承诺让所有参与者都能感受到尊重和包容，无论年龄、体型、肤色、能力、教育程度、性别认同、经验水平、国籍、个人形象、种族、宗教或性取向。

## 我们的标准

### ✅ 鼓励的行为

- 使用欢迎和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注社区共同利益
- 对其他社区成员表现出同理心

### ❌ 不可接受的行为

- 使用带有性暗示的语言或图像
- 人身攻击或贬低性评论
- 公开或私下骚扰
- 未经授权发布他人的私人信息
- 其他不道德、不专业的行为

## 适用场景

本行为准则适用于所有社区场景，包括：
- GitHub Issues和Discussions
- 官方论坛和交流群
- 线下活动
- 代表社区的公开场合

## 执行

社区管理者有责任解释和执行本准则。对于违反本准则的行为，将采取以下措施：

1. **口头警告** — 首次轻微违规
2. **书面警告** — 重复或较严重违规
3. **暂时封禁** — 严重违规
4. **永久除名** — 反复严重违规或严重不当行为

# 安全政策

## 报告安全漏洞

我们非常重视安全问题。如果您发现了安全漏洞，请通过以下方式报告：

### ⚠️ 报告方式

**请勿**在GitHub Issues中公开报告安全漏洞！

请发送邮件至：3978003571@qq.com

### 📋 报告内容

请在报告中包含：

1. **漏洞描述** — 清楚地描述漏洞类型
2. **复现步骤** — 如何重现该漏洞
3. **影响评估** — 漏洞可能造成的危害
4. **建议修复** — 您对修复方案的建议（可选）

### ⏱️ 响应时间

- **初步响应**：72小时内
- **漏洞确认**：7天内
- **修复计划**：30天内（严重漏洞将更快处理）

### 🎁 感谢

我们会公开感谢所有帮助我们提升安全性的研究者。
